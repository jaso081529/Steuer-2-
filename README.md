# Steuer 2

Fertige Web-App für Einnahmen, Ausgaben, Belege, CSV-Export, JSON-Backup und Druckübersicht.

## GitHub Pages über Actions öffnen

1. Repository `Steuer-2` auf GitHub erstellen.
2. Alle Dateien aus diesem ZIP in den Branch `main` hochladen.
3. In GitHub auf **Actions** gehen.
4. Workflow **Deploy Steuer 2 Web App** öffnen.
5. Nach erfolgreichem Lauf auf den angezeigten GitHub-Pages-Link gehen.

## Dateien

- `index.html` – komplette App in einer Datei
- `.github/workflows/pages.yml` – GitHub Actions Deployment
- `README.md` – Kurzanleitung
